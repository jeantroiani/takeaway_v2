Takeaway
===========

A object oriented model of a restaurant takeaway application. Tested with Rspec. Also uses Twilio to send message to customers. 


How to use it
=============
git clone https://github.com/jeantroiani/takeaway_v2.git


Technologies used
=============
- Ruby
- Rspec
- CodeClimate
- Twilio
